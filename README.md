# IMDb Web Scraping Project

## Overview
This project demonstrates web scraping of IMDb title data using Python and BeautifulSoup.  
The scraped data is cleaned, structured, and stored in a CSV file ready for analysis.

---

## Tools & Technologies
- Python  
- Requests  
- BeautifulSoup (bs4)  
- CSV / Excel  
- Jupyter Notebook  

---

## Data Extracted
- Title name  
- Release year (start year extracted from ranges)   
- IMDb rating (if available)  

---

## Data Handling
- Handled missing values safely (`N/A`)
- Cleaned encoding issues in release year values
- Ensured proper CSV headers and structure

---

## Learning Outcomes
- Real-world web scraping experience  
- Handling inconsistent HTML structures  
- Cleaning messy scraped data  
- Preparing datasets for data analysis
